Данный скрипт "TEST_2_0" позволяет анализировать данные о производительности сотрудников из CSV-файлов и формировать отчет, показывающий среднюю производительность по каждой должности.

В проекте есть readme, .gitignore. В проекте есть файл с зависимостями, линтер и форматтер. Проект разбит по модулям, вся функциональность в разных файлах. Используется контекстный менеджер.

Используются аннотации. Для валидации названия отчетов используются choices в argparse. Используется ООП. Используются абстрактные классы и протоколы, с их помощью описываются отчёты. 

Используется dataclasses, DTO, TypedDict. Используется @pytest.fixture. Используется @pytest.mark.parametrize.

Для корректной работы проекта необходим запуск requirements.txt:

requirements.txt содержит в себе:

tabulate~=0.9.0

pytest~=7.4.0

flake8~=6.0.0

black~=23.7.0

Команда для установки зависимостей:

pip install -r requirements.txt

Создание и запуск виртуального окружения:

python -m venv venv — создание виртуального окружения.

venv\Scripts\activate — запуск виртуального окружения.

Запуск программы:

python main.py --files employees1.csv employees2.csv --report report.txt

Где:

--files - аргумент, указывающий входные CSV-файлы.

--report — аргумент, указывающий выходной файл отчёта. Если не указано, отчёт выводится в консоль.

Вызов справки:

python main.py --help

Запуск тестов из корня проекта:

pytest tests/
